About This File

This is the canonical digital style reference for Burning Man Project (burningman.org). It documents design tokens — colors, typography, and component styles — extracted from the live site's theme. SaaS services such as Typeform, Constant Contact, Mailchimp, and Canva can use this page to import brand styles automatically.

For developers: The :root { } block in this file's <style> tag contains all design tokens as CSS custom properties prefixed with --bmp-. Point your import tool at this file's URL, or copy the root block into your project's CSS.

Links

Inline links use Purple 400 (#534BE7) with underline decoration. On hover, color deepens to Purple 600 (#181287). Link contrast on white meets WCAG AA (4.5:1 minimum).

Buttons & Interactive Elements

All button variants use font-weight 700, border-radius 4px, and padding of ~10px 20px. Hover styles are shown in parentheses.

Default bg #141414 → hover #534BE7
Alternative bg #2018B4 → hover #534BE7
Outline border #534BE7 → hover bg #534BE7
Subtle border #7E78ED → hover bg #D4D2F9
Accessibility: All button variants meet WCAG AA contrast requirements. Default (#FFFFFF on #141414) = 19.1:1. Outline (#534BE7 on #FFFFFF) = 6.6:1. Subtle (#181287 on #FFFFFF) = 12.1:1. Always include visible focus styles (ring shown on keyboard nav).
